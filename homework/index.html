Flight Departure Information

GET 'localhost:8080' : 항공 출발 정보 이용 방법 소개

GET 'localhost:8080/flight' : 전체 항공 출발 정보 검색

GET 'localhost:8080/flights?to=Paris' : Paris로 가는 항공 출발 정보 검색

POST 'localhost:8080/flight' : 새 항공 출발 정보 생성

PUT 'localhost:8080/flight/2' : 2번 id의 항공 출발 정보 수정

DELETE 'localhost:8080/flight/2' : 2번 id의 항공 출발 정보 삭제
